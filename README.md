Tổng quan dự án
Dự án này là một hệ thống mở rộng hiển thị và cảnh báo chuyên dụng cho thiết bị giám sát
hành trình NASA-4G. Sử dụng vi điều khiển ESP32-C3 và hệ điều hành thời gian thực
FreeRTOS, hệ thống xử lý dữ liệu từ GPS Gateway qua RS232 để cung cấp thông tin trực quan
và cảnh báo giọng nói cho tài xế.
